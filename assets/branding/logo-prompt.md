# Novelist Logo Generation Prompt

Use the following prompt with an AI image generator (Midjourney, DALL-E, Stable Diffusion):

---

## Primary Prompt

> A minimal, modern app icon for a desktop writing application called "Novelist". The icon features an elegant quill pen nib or fountain pen tip, formed by a single continuous flowing stroke, resting on or emerging from an open book or paper sheet. The design uses a deep indigo/navy blue (#1e293b) as the primary color with a warm gold/amber (#f59e0b) accent on the pen nib. Clean geometric style with subtle rounded corners, suitable for macOS app icon at 1024x1024. White or very light background. No text, no letters, no gradients — flat design with at most 3 colors. The feel should be: professional, literary, calm, focused.

## Variations

### Variation A — Minimal Letter Mark
> A single stylized letter "N" designed as a macOS app icon. The "N" is formed by two vertical strokes connected by a diagonal, where the diagonal resembles a flowing pen stroke with a slight calligraphic taper. Deep slate blue (#334155) on a soft cream/paper-white (#fafaf9) background. 1024x1024, flat design, rounded square icon shape. No other elements.

### Variation B — Book + Pen Combined
> A macOS app icon showing a side view of an open book where the right page curls upward and transforms into a fountain pen nib pointing up-right. Single continuous line art style. Indigo (#4f46e5) line on white background. Minimal, elegant, 1024x1024 flat vector style. Rounded square shape.

### Variation C — Typewriter Key
> A macOS app icon of a single vintage typewriter key viewed from above, with the letter "N" on it. Soft shadow beneath. Cream key surface, dark charcoal (#1e1e1e) letter, subtle brass ring around the key. 1024x1024, photorealistic with slight depth. Rounded square.

## Design Guidelines

- **Shape**: macOS rounded-square (squircle) format
- **Size**: 1024x1024 master, export 512, 256, 128, 32
- **Colors**: Max 3. Prefer deep blues, warm accents, paper tones
- **Style**: Flat or minimal depth. No heavy gradients, no 3D
- **Feel**: Literary, professional, focused, calm — like a premium writing tool
- **Avoid**: Generic document icons, pencil emoji style, clip art, busy compositions
- **Reference apps**: Typora (simple T mark), iA Writer (monospace cursor), Ulysses (butterfly)
