## Key Design Goals for Novelist

1. Ultra-lightweight client (<30MB)
2. WYSIWYG/near-WYSIWYG markdown editing
3. Plugin architecture (whiteboard, mind-map, etc. as plugins)
4. Project-level management (multi-project, open-directory model)
5. External tool friendly (plain markdown, no proprietary formats)
6. Project-specific formatting/export config
7. Target users: academic writers, screenwriters, web novelists, educators
8. Zen mode
9. High performance (handle 10MB+ files without lag)
