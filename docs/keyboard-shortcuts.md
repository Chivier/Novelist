# Keyboard Shortcuts

All shortcuts use `Cmd` on macOS and `Ctrl` on Windows/Linux. Shortcuts are customizable via Settings > Shortcuts.

## General

| Shortcut | Action |
|----------|--------|
| Cmd+S | Save file |
| Cmd+N | New file |
| Cmd+W | Close tab |
| Cmd+, | Settings |
| Cmd+Shift+P | Command palette |

## Navigation

| Shortcut | Action |
|----------|--------|
| Cmd+G | Go to line |
| Cmd+Click | Open link in browser |

## Panels

| Shortcut | Action |
|----------|--------|
| Cmd+B | Toggle sidebar |
| Cmd+Shift+O | Toggle outline |
| Cmd+Shift+D | Toggle draft note |
| Cmd+\\ | Toggle split view |

## Writing

| Shortcut | Action |
|----------|--------|
| F11 | Toggle Zen Mode |
| Escape | Exit Zen Mode |
